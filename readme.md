## Unity8 for desktops

*NOTE* this is still under development, expect bugs!

Also this will probably *NOT* work with proprietary drivers

### How to install

1. Clone this repo
2. Run the install script

```
./install.sh
```
3. Logout, select "unity8" and enjoy :)
